 My Portfolio

Description

This repository contains the code for a personal portfolio website. The website showcases the work and skills of Luto Avela Mfabane, a web developer. It includes sections for a profile, about information, experience, projects, and contact details.

Project Structure

The project consists of the following main files:

- `index.html` - The main HTML file that structures the content of the portfolio.
- `style.css` - The primary stylesheet that styles the webpage.
- `mediaqueries.css` - Stylesheet that includes media queries for responsive design.
- `script.js` - JavaScript file for interactive elements.

HTML Structure

`index.html`

The HTML structure includes:

- **Navigation (`nav`):** A top navigation bar with links to different sections of the page.
- **Profile Section (`#profile`):** Displays the profile picture, name, and brief introduction along with social media links and buttons.
- **About Section (`#about`):** Provides details about the developerâ€™s background and education.
- **Experience Section (`#experience`):** Highlights programming skills and expertise.
- **Projects Section (`#projects`):** Showcases recent projects with links to their GitHub repositories and live demos.
- **Contact Section (`#contact`):** Provides contact information including email and phone number.
- **Footer:** Contains navigation links and copyright information.

Media Queries

Responsive design is handled through media queries in `mediaqueries.css` to ensure the portfolio looks good on different devices.

 CSS Styles

 `style.css`

- Global Styles:** Basic styles for body, text, and links.
- Navigation Styles:** Styling for the navigation bar, including hover effects.
- Section Styles:** Styles for each section to define layout and spacing.
- Profile Section:** Flexbox layout for profile picture and text alignment.
- About Section:** Layout for about details and icons.
- Projects Section:** Styles for project cards and buttons.
- Contact Section:** Styling for contact information display.

 `mediaqueries.css`

Media queries adjust the layout for different screen sizes:

- For screens up to 1400px: Adjusts profile height and section layout.
- For screens up to 1200px: Hides desktop navigation and shows hamburger menu, adjusts section margins.
- For screens up to 600px: Adjusts text sizes, section heights, and layout for smaller devices.

JavaScript

`script.js`

The JavaScript file is used for:

- Opening the CV PDF on button click.
- Redirecting to sections or external links on button or icon click.
